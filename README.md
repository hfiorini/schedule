# Simple Scheduling System

Requirements:
- Java 17
- Maven 3.9.6
- Docker 26.1.1

To run this app:
- Run `mvn clean install`
- Then `docker-compose build && docker-compose up`
- After app initialization, you'll be able to use Swagger at this URL:
`http://localhost:9000/swagger-ui/index.html#/`
